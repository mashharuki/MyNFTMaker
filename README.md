# MyNFTMaker
コンテンツを分散型ストレージにアップロードしてNFTを作成するアプリのリポジトリです。

### デプロイ用のコマンド
 1. `npx hardhat run scripts/run.js`
 2. `npx hardhat run scripts/deploy.js`
